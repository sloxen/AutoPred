AutoPred is a lightweight predictive modelling layer for small and medium-sized businesses.

It transforms raw operational data into clear forecasts and decision-ready insights through simple, explainable models and clean visualisations.

Find more on [AutoPred website](https://autopred.sloxen.com).

# Features

AutoPred helps answer practical business questions:

- How much will we sell tomorrow?
- Which products are predictable?
- When does demand change?
- How much better is our model than a simple baseline?

It focuses on clarity, not complexity.

---

# Key Capabilities

- Structured train, validation, and test workflow  
- Baseline comparison using dummy models  
- Multiple model training and selection  
- Exportable prediction datasets  
- Business-friendly visualisations  
- Ready for web-based dashboards

# Analysis

AutoPred generates structured outputs:

- scored datasets  
- model comparison tables  
- final metrics  
- visualisation-ready data  

These outputs can be directly used in dashboards or applications. AutoPred visualisations are designed to be:

- simple to read  
- focused on key insights  
- suitable for non-technical users  

They highlight:

- actual vs predicted trends  
- model improvement over baseline  
- product-level prediction accuracy

# Roadmap

AutoPred is evolving into a web-based service.

Planned features:

- interactive dashboards  
- drag-and-drop data upload  
- model selection in browser  
- real-time forecasting  
- API access  

The [bakery demo](https://github.com/sloxen/AutoPred/blob/main/bakery_demo.md) is an early instance of this system.
